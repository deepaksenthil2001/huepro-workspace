#  HuePro Workspace - Enterprise Dashboard

A modern, premium full-stack application designed for efficient management of projects, contractors, workers, and materials. Built with a strict focus on "Apple-standard" UI/UX guidelines and a robust, scalable backend architecture.

##  Tech Stack

### Frontend Architecture
* **Framework:** React.js (via Vite for blazing fast builds)
* **Styling:** Tailwind CSS (Custom glassmorphism & gradients)
* **Animations:** Framer Motion (Fluid, premium transitions)
* **Icons:** Lucide React
* **State Management & API:** React Hooks, Axios

### Backend Architecture
* **Core:** Java 17, Spring Boot 3.x
* **Security:** Spring Security (Secure Auth API)
* **Database Management:** Spring Data JPA (Hibernate)
* **Database:** Relational Database (PostgreSQL / MySQL)
* **Architecture Pattern:** MVC (Model-View-Controller)

##  Key Features
*  **Secure Authentication:** Robust login, registration, and session management.
*  **Interactive Dashboard:** Live activity feeds, statistics, and business overviews.
*  **Premium Profile Management:** Real-time profile updates using advanced glassmorphic modals.
*  **Resource Tracking:** Dedicated modules for Projects, Workers, Materials, and Invoices.
*  **Fully Responsive:** Optimized for both desktop and mobile experiences.

##  Repository Structure (Monorepo)
This repository is structured as a Monorepo containing both the client and server applications:
* `/FrontEnd` - Contains the React/Vite UI codebase.
* `/BackEnd` - Contains the Spring Boot REST API codebase.
